# Cowsay

ASCII art avatars emote your messages, now with the cute sign-holding bunny from Twitter

```
 _______________
| TRANS RIGHTS! |
 ---------------
(\__/) ||
(•ㅅ•) ||
/ 　 づ
```

## Installation

    git clone https://github.com/queenkjuul/cowsay.git
    gem install cowsay/pkg/cowsay-0.3.0.gem

And then execute:

    $ cowsay 'Hello world!'
